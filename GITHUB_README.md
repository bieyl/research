## 📖 项目简介

**学术龙虾 v2** 是一款面向科研工作者的 AI 研究助手，采用本地优先架构保障数据安全，同时集成 arXiv 实时论文抓取，帮助研究者高效获取领域经典与前沿进展。

**核心理念：** 让知识产生连接

---

## ✨ 核心功能

### 1. 🌳 知识图谱

构建论文与实验的关联网络，支持树状/网络视图。

- **智能关联** - 基于关键词、方法、领域的自动关联
- **去重逻辑** - 标题/目的自动去重
- **可视化** - 树状视图清晰展示知识结构

### 2. 💡 智能推荐

**三轨制推荐策略** - 经典 + 前沿 + 实时

| 类型 | 标识 | 年份 | 说明 |
|------|------|------|------|
| 📜 经典奠基 | 橙色 | 2012-2017 | 领域开山之作 |
| 🆕 最新进展 | 绿色 | 2018-2023 | 前沿 SOTA |
| 🌐 arXiv 实时 | 粉色 | 实时 | 最新预印本 |

**覆盖领域：** Transformer、CNN、BERT、注意力机制、ResNet 等

### 3. ⚡ 文献摘要

5 秒读懂一篇论文，输出结构化摘要：

- 【研究问题】- 论文解决的核心问题
- 【研究方法】- 采用的技术方法
- 【核心结论】- 主要研究发现
- 【创新点】- 与已有工作的区别
- 【关键词】- 3-5 个核心关键词

### 4. 📊 实验日志

自动生成结构化实验报告：

- 【实验目的】- 实验要解决的问题
- 【实验条件】- 温度、压力、时间等参数
- 【实验结果】- 结构化表格呈现
- 【结果分析】- 趋势分析、异常值识别
- 【后续建议】- 优化方向与下一步实验建议

### 5. 📑 PPT 大纲

一键生成学术汇报 PPT 框架：

1. 研究背景与意义
2. 相关工作
3. 方法与技术路线
4. 实验设计与结果
5. 结论与展望
6. 参考文献

### 6. 📖 参考文献格式化

支持 GB/T 7714、APA、MLA、IEEE、BibTeX 等主流格式。

## 🚀 快速开始

### 安装

```bash
# 1. 克隆项目
git clone https://github.com/bieyunlong1/academic-lobster-v2.git
cd academic-lobster-v2

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动服务
```

## 🎬 使用场景

### 场景 1：文献调研

**问题：** 刚进入新领域，不知道从哪些论文开始读？

**解决方案：**
1. 输入领域关键词（如"Transformer 图像分类"）
2. 获取推荐论文（经典 + 前沿 + 实时）
3. 阅读经典论文建立知识框架
4. 追踪最新研究把握前沿方向

**效果：** 1 小时完成原本需要 1 周的文献调研

---

### 场景 2：组会汇报

**问题：** 明天组会，今晚才开始准备 PPT？

**解决方案：**
1. 输入研究主题
2. 一键生成 PPT 大纲
3. 填充具体内容

**效果：** 10 分钟完成 PPT 框架

---

### 场景 3：实验记录

**问题：** 实验数据散乱，难以整理成规范报告？

**解决方案：**
1. 输入实验数据
2. 自动生成结构化实验报告
3. 包含结果分析与后续建议

**效果：** 5 分钟完成原本需要 1 小时的实验记录
