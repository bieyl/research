## 📋 测试方法论

### 测试环境

```
Hardware: MacBook Pro M1, 16GB RAM, 512GB SSD
Software: macOS 14.0, Python 3.11
Dataset: 10,000 篇计算机视觉论文 (2015-2026)
Network: 100Mbps 宽带
```

### 测试指标

- **搜索延迟:** 从输入查询到返回结果的时间
- **推荐延迟:** 从请求到返回推荐的时间
- **PPT 生成时间:** 从点击到生成的时间
- **内存占用:** 运行时的 RAM 使用量
- **磁盘占用:** 安装 + 数据文件大小

### 测试工具

- **性能测试:** pytest-benchmark
- **负载测试:** locust
- **内存分析:** memory_profiler
- **网络分析:** Wireshark

## 🔬 可复现性

### 复现步骤

```bash
# 1. 克隆仓库
git clone https://github.com/bieyl/research.git
cd research/academic-lobster

# 2. 安装依赖
pip install -r requirements.txt
pip install -r requirements-dev.txt

# 3. 运行基准测试
pytest tests/benchmarks/ -v

# 4. 生成报告
pytest --benchmark-json=benchmark.json tests/benchmarks/
python scripts/generate_benchmark_report.py
```
