## 快速入门

### 第一步：启动程序

```bash
python src/main.py
```

启动后将看到主界面，包含 5 个核心功能模块。

### 第二步：选择功能

| 模块 | 功能 | 快捷键 |
|------|------|--------|
| 文献摘要 | 上传 PDF 生成摘要 | Ctrl+1 |
| 关键词提取 | 提取关键词并分类 | Ctrl+2 |
| 实验日志 | 录入数据生成报告 | Ctrl+3 |
| PPT 大纲 | 生成组会汇报结构 | Ctrl+4 |
| 参考文献 | 格式化参考文献 | Ctrl+5 |

### 第三步：开始使用

点击对应模块，按提示操作即可。

## 功能详解

### 1. 文献智能摘要

**用途：** 快速了解论文核心内容，判断是否值得精读

**操作步骤：**

1. 点击"文献摘要"模块
2. 拖入 PDF 文件或点击"选择文件"
3. 点击"生成摘要"按钮
4. 等待 30 秒，查看结构化摘要

**输出内容：**

```
【研究问题】xxx
【研究方法】xxx
【核心结论】xxx
【创新点】xxx
【关键词】xxx, xxx, xxx
```

**支持格式：** PDF, TXT, DOCX

---

### 2. 关键词自动提取

**用途：** 快速提取文献核心关键词，自动学科分类

**操作步骤：**

1. 点击"关键词提取"模块
2. 输入文本或上传文档
3. 点击"提取关键词"
4. 查看关键词列表和学科分类

**输出内容：**

```
关键词（5-10 个）：
- 卷积神经网络
- 医学影像
- 病灶检测
- 迁移学习
- 数据增强

学科分类：
计算机应用 → 医学 AI → 影像诊断
```

---

### 3. 实验日志助手

**用途：** 快速录入实验数据，自动生成规范报告

**操作步骤：**

1. 点击"实验日志"模块
2. 选择输入方式（文本/语音）
3. 输入原始实验数据
4. 点击"生成日报"
5. 查看并导出实验报告

**输入示例：**

```
2026-03-10 实验 3 组：
温度 60°C/70°C/80°C
反应时间 2h
产物收率 45%/62%/38%
```

**输出示例：**

```markdown
# 实验日报 2026-03-10

## 实验目的
探究温度对反应收率的影响

## 实验数据

| 组别 | 温度 (°C) | 时间 (h) | 收率 (%) |
|------|-----------|----------|----------|
| 1    | 60        | 2        | 45       |
| 2    | 70        | 2        | 62       |
| 3    | 80        | 2        | 38       |

## 数据分析
- 70°C 时收率最高 (62%)
- 80°C 时收率下降，可能出现副反应

## 建议
后续优化 65-75°C 温度区间
```

**导出格式：** Markdown, PDF, Word

### 4. 组会 PPT 大纲

**用途：** 快速生成组会汇报结构，节省准备时间

**操作步骤：**

1. 点击"PPT 大纲"模块
2. 输入本周研究进展
3. 点击"生成大纲"
4. 查看 PPT 结构和每页要点

**输入示例：**

```
本周完成了 ResNet-50 在自定义数据集上的迁移学习，
准确率 87%，遇到小样本过拟合问题。
```

**输出示例：**

```
P1 标题页
   - 研究进展汇报
   - 姓名、日期

P2 研究背景 (1 页)
   - 问题定义
   - 研究意义

P3 相关工作 (2 页)
   - 方法 A
   - 方法 B

P4 方法设计 (2 页)
   - 整体架构
   - 关键改进

P5 实验结果 (3 页)
   - 数据集介绍
   - 对比实验
   - 消融实验

P6 问题分析 (1 页)
   - 过拟合现象
   - 可能原因

P7 下一步计划 (1 页)
   - 数据增强
   - 正则化优化

P8 致谢 (1 页)
```

### 5. 参考文献格式化

**用途：** 一键转换参考文献格式，避免格式错误

**操作步骤：**

1. 点击"参考文献"模块
2. 输入杂乱的参考文献信息
3. 选择目标格式（GB/T 7714 / APA / IEEE）
4. 点击"格式化"
5. 复制结果到论文

**输入示例：**

```
He K, Zhang X, Ren S, Sun J. Deep Residual Learning 
for Image Recognition. CVPR 2016.
```

**输出示例：**

```
【GB/T 7714】
[1] HE K, ZHANG X, REN S, et al. Deep Residual Learning 
for Image Recognition[C]//CVPR, 2016.

【APA】
He, K., Zhang, X., Ren, S., & Sun, J. (2016). 
Deep Residual Learning for Image Recognition. CVPR.

【IEEE】
[1] K. He et al., "Deep Residual Learning for Image 
Recognition," CVPR, 2016.
```

**支持格式：** GB/T 7714, APA, IEEE, MLA, Chicago

## 常见问题

### Q1: 上传的 PDF 会被上传到云端吗？

**A:** 不会。所有文件均在本地处理，数据永不离开您的设备。

### Q2: 语音识别需要联网吗？

**A:** 可选。支持离线语音识别（需下载语言包），也支持在线模式（精度更高）。

### Q3: 支持中文论文吗？

**A:** 支持。学术龙虾针对中英文论文均做了优化。

### Q4: 实验数据会保存多久？

**A:** 默认永久保存，您可随时手动删除。所有数据存储在本地 SQLite 数据库。

### Q5: 如何导出所有数据？

**A:** 点击"设置" → "数据管理" → "导出全部数据"，可选择导出为 ZIP 包。

## 最佳实践

### 文献管理

1. **批量处理**：一次性拖入多篇 PDF，学术龙虾会依次处理
2. **标签分类**：为每篇文献添加标签（如"精读"、"略读"、"引用"）
3. **定期整理**：每周花 10 分钟整理文献库，删除不相关的

### 实验记录

1. **及时录入**：实验完成后立即录入，避免遗忘
2. **语音输入**：戴手套时可用语音输入，解放双手
3. **定期备份**：每周导出实验日志，防止数据丢失

### 组会准备

1. **提前生成**：组会前 1 天生成 PPT 大纲
2. **迭代优化**：根据导师反馈调整大纲
3. **积累模板**：保存优秀的大纲作为模板
