# 学术龙虾 v2 - 智能推荐策略

## 推荐理念

**"一手抓经典，一手抓前沿"**

智能推荐功能采用**双轨制推荐策略**：
- 📜 **经典奠基性论文** - 领域开山之作，必读基础
- 🆕 **最新研究进展** - 前沿 SOTA，把握方向

## 推荐算法

### 评分规则

1. **经典论文基础分**：95-98 分
2. **最新论文基础分**：85-92 分
3. **关键词匹配**：+0-20 分
4. **方法匹配**：+0-15 分
5. **问题匹配**：+0-10 分

### 排序策略

1. 按分数降序排列
2. 经典论文优先展示（分数略高）
3. 最新研究紧随其后
