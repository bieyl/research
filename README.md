## 📋 核心功能

### 功能 1：科研知识图谱（文字树状）

**输入：** 论文标题 + 研究问题 + 方法 + 结论

**输出：**
```
🌳 科研知识图谱

【研究问题】深层网络训练退化问题
├─【方法】残差连接
│  ├─📄 ResNet (He et al., 2016)
│  └─📄 Identity Mappings (He et al., 2016)
└─【方法】改进的残差块
   └─🧪 你的实验 (2026-03-12)
```

**禁止：** ❌ 复杂网络图 ❌ 违规爬取 ❌ 外部调用

---

### 功能 2：实验 - 文献智能关联（双模式）

#### 基础模式（默认，本地）
- 仅关联本地上传的自有文献
- 无网络也能用

#### 进阶模式（用户主动触发，合规联网）
- 合规爬取公开学术平台（arXiv/PubMed/知网公开摘要）
- 明确标注来源，尊重知识产权
- 爬取内容仅用于临时推荐，不本地存储

**输出：**
```
💡 智能推荐

📚 关联文献（按相关性排序）：

1. ResNet 原始论文 (He et al., 2016)
   相关性：★★★★★ (95 分)
   匹配理由：你的实验基于该论文的残差结构
   来源：本地知识库

🌐 公开学术平台补充推荐：

2. ECA-Net (Wang et al., CVPR 2020)
   来源：arXiv (公开摘要)
   相关性：★★★☆☆ (78 分)
   链接：https://arxiv.org/abs/1910.03151
   ⚠️ 声明：仅获取公开摘要，未存储全文
```

---

### 功能 3：组会 PPT 大纲 + 讲稿一键生成

**输入：** 本周实验 + 相关文献

**输出：** 标准 5 页组会 PPT 大纲 + 可直接朗读的讲稿

**不做：** ❌ 复杂排版 ❌ 图片生成

**只做：** ✅ 可直接复制到 PPT 的文本内容 ✅ 可直接朗读的讲稿

---

### 功能 4：本地安全沙箱模式

**规则：**
- ✅ 所有自有数据仅本地处理、不上传云端
- ✅ 联网爬取仅临时获取公开摘要，用完即清理
- ✅ 明确标注行为边界

## 🏆 参赛材料

### 产品介绍（150 字）

> 学术龙虾 v2 是一款科研本地知识大脑，专为研究生、青年教师设计。核心功能包括：知识图谱（自动构建"问题→方法→论文→实验"知识树）、实验 - 文献智能关联（本地 + 合规联网双模式）、组会 PPT 大纲 + 讲稿一键生成。所有自有数据本地存储，不上传云端；联网爬取仅获取公开摘要，用完即清理。目标是让碎片化的科研知识形成体系，让重复劳动自动化，让研究者专注于真正的创新。

### 30 秒路演脚本

> **（痛点）** 在座的评委老师可能都带过研究生。您有没有遇到过这种情况：学生读了两年论文，但知识是碎片化的，问他这个领域有哪些方法，他说不清楚。
> 
> **（解决方案）** 学术龙虾 v2 不是又一个文献管理工具，而是科研知识的大脑。它自动构建"问题→方法→论文→实验"知识树，让实验和文献智能关联，一键生成组会 PPT 和讲稿。
> 
> **（价值）** 所有数据本地存储，安全可控；联网爬取合规透明，尊重知识产权。我们的愿景：让每一位研究者，都拥有平等的科研智能支持。

### 5 分钟演示流程

| 时间 | 环节 | 内容 |
|------|------|------|
| 0:00-0:30 | 开场 | 痛点 + 定位 |
| 0:30-2:00 | 功能 1 | 知识图谱（添加 2 篇论文 +1 个实验） |
| 2:00-3:00 | 功能 2 | 智能推荐（本地 + 联网双模式） |
| 3:00-4:00 | 功能 3 | PPT 一键生成 |
| 4:00-5:00 | 总结 | 价值 + Q&A |
