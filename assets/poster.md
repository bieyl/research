# 🦞 学术龙虾 v2
## 科研知识大脑

---

### 核心定位
**课题组私有、本地优先、可落地的科研知识大脑**

---

### 差异化壁垒（独家功能）

| 功能 | 市场现状 | 我们的优势 |
|------|---------|-----------|
| 🌳 知识树 | 无同类产品 | 独家知识图谱 |
| 💡 实验↔文献关联 | 无同类产品 | 本地 + 合规联网双模式 |
| 📊 PPT+ 讲稿生成 | 通用 PPT 工具不懂科研 | 科研场景专用 |
| 🔒 全程本地运行 | 云端工具数据不可控 | 安全可审计 |

---

### 核心功能

#### 1️⃣ 科研知识图谱
- 树状视图清晰展示论文库与实验库
- 基于关键词、方法、领域的自动关联
- 标题/目的自动去重

#### 2️⃣ 智能推荐（三轨制）
- 📜 **经典奠基** (2012-2017) - 领域开山之作
- 🆕 **最新进展** (2018-2023) - 前沿 SOTA
- 🌐 **arXiv 实时** - 最新预印本

#### 3️⃣ 文献摘要
- 自动提取论文核心信息
- 结构化输出：问题/方法/结论

#### 4️⃣ 实验日志
- 自动生成结构化实验报告
- 支持多种实验类型

#### 5️⃣ PPT 大纲
- 一键生成学术汇报框架
- 附带可直接朗读的讲稿

#### 6️⃣ 参考文献格式化
- 支持 GB/T 7714、APA、MLA 等主流格式

---

### 技术亮点

✅ **本地优先架构** - SQLite 本地存储，数据不出实验室  
✅ **arXiv API 集成** - 实时获取最新研究  
✅ **高端学术 UI** - 渐变配色、卡片式布局、响应式设计  
✅ **中文场景优化** - GB/T 7714 格式、中文论文结构  

---

### 适用场景

📚 **文献调研** - 快速了解领域全貌  
🔬 **实验记录** - 结构化保存实验数据  
📊 **学术汇报** - 一键生成 PPT 大纲  
✍️ **论文写作** - 参考文献自动格式化  

---

### 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动 Web 界面
python src/web_app_v3.py

# 访问
http://localhost:5001
```
