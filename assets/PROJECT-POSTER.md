## 📋 核心功能

### 1️⃣ 科研知识图谱

- 树状视图清晰展示论文库与实验库
- 基于关键词、方法、领域的自动关联
- 标题/目的自动去重

```
【研究问题】深层网络训练退化问题
├─【方法】残差连接
│  ├─📄 ResNet (He et al., 2016)
│  └─📄 Identity Mappings (He et al., 2016)
└─【方法】改进的残差块
   └─🧪 你的实验 (2026-03-16)
```

### 2️⃣ 智能推荐（双模式）

- 📜 **基础模式** — 仅本地知识库，离线可用
- 🌐 **进阶模式** — 合规联网搜索（arXiv/PubMed/CNKI）

```
💡 智能推荐

📚 关联文献（按相关性排序）：

1. ResNet 原始论文 (He et al., 2016)
   相关性：★★★★★ (95 分)
   来源：本地知识库

2. ECA-Net (Wang et al., CVPR 2020)
   来源：arXiv (公开摘要)
   相关性：★★★☆☆ (78 分)
   ⚠️ 仅获取公开摘要，未存储全文
```

### 3️⃣ 文献摘要

- 自动提取论文核心信息
- 结构化输出：问题/方法/结论
- 30 秒/篇，效率提升 240 倍

### 4️⃣ 实验日志

- 语音输入原始数据
- 自动生成结构化实验报告
- 支持多种实验类型

### 5️⃣ PPT 大纲

- 一键生成学术汇报框架
- 附带可直接朗读的讲稿
- 5 页标准组会格式

### 6️⃣ 参考文献格式化

- 支持 GB/T 7714、APA、MLA 等主流格式
- 5 秒/篇，效率提升 180 倍

## 🚀 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/bieyl/research.git
cd research

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动 Web 界面
python3 src/web_app_v2.py

# 4. 访问
# http://localhost:5001
```

**Docker 部署：**
```bash
docker build -t academic-lobster .
docker run -d -p 5001:5001 -v ./data:/app/data academic-lobster
```

## 🎨 海报设计说明

- **主色调：** 龙虾红 (#C41E3A) + 学术蓝 (#1E3A8A)
