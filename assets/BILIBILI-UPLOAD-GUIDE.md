# 📤 B 站视频上传完整指南

## 🎬 快速录制方案（15 分钟）

### 方案 A：手机录制电脑屏幕（最快）

1. **用手机拍摄电脑屏幕**
   - 手机横屏
   - 1080p 分辨率
   - 稳定持握或使用支架

2. **操作流程：**
   - 打开 http://localhost:5001
   - 按功能逐个展示（每个 20-30 秒）
   - 总时长 2-3 分钟

### 方案 B：OBS 录屏（推荐）

1. **下载 OBS：** https://obsproject.com/

2. **设置：**
   - 视频 → 画布：1920x1080
   - 视频 → 输出：1920x1080, 30fps
   - 输出 → 录制：MP4 格式

3. **录制：**
   - 点击"开始录制"
   - 演示所有功能
   - 点击"停止录制"
