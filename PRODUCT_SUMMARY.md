## 一、产品定位重构

### 从"工具"到"大脑"

**v1 定位：** 科研全流程 AI 助手（工具集合）

**v2 定位：** 科研知识大脑（知识管理 + 智能关联）

**为什么重构：**
1. 市面上已有太多"工具"，但缺少"知识管理"
2. 科研工作者的核心痛点不是"没有工具"，而是"知识碎片化"
3. 差异化竞争：不做另一个 Zotero/ChatPDF，做知识图谱

---

## 二、核心差异化功能

### 1. 🌳 科研知识图谱（独家）

**解决的问题：**
> 研究生读了 3 年论文，毕业时说不清楚这个领域有哪些研究方向。

**功能：**
- 自动构建"问题→方法→结论"三层知识树
- 基于关键词、方法相似度自动建立关联
- 可视化展示知识网络

**技术实现：**
- `src/knowledge_graph.py` - 知识图谱核心
- 自动提取三元组（问题、方法、结论）
- 相似度算法建立关联

**演示：**
```python
# 添加论文后自动构建知识树
kg.add_paper({
    'title': 'Deep Residual Learning...',
    'research_question': '深层网络训练退化问题',
    'method': '残差连接',
    'keywords': ['深度学习', '图像识别']
})

# 查看知识地图
print(kg.export_to_text())
```

---

### 2. 💡 实验 - 文献智能关联（独家）

**解决的问题：**
> 写论文时，忘记哪篇论文支持这个实验结果。

**功能：**
- 录入实验时，自动推荐相关论文
- 阅读论文时，自动推荐相关实验
- 生成引用建议

**技术实现：**
- `src/smart_recommend.py` - 智能推荐引擎
- 关键词提取 + 相似度匹配
- 关联原因说明

**演示：**
```python
# 为实验推荐论文
recommender.recommend_papers_for_experiment(
    "使用 ResNet-50 进行图像分类，引入注意力机制"
)
# 输出：ResNet 原始论文、Identity Mappings、SENet 等
```

---

### 3. 📊 组会 PPT 一键生成（独家）

**解决的问题：**
> 每周组会花 2-3 小时准备 PPT，但内容都是已有的实验记录和论文。

**功能：**
- 选择时间范围，自动聚合实验和论文
- 生成完整 PPT（含图表、数据对比）
- 导出为 PPTX/PDF

**技术实现：**
- `src/ppt_outline.py` - PPT 大纲生成
- 聚合指定时间范围内的数据
- 填充 PPT 模板（python-pptx）

**演示：**
```python
# 生成 PPT
outline = ppt_outliner.generate(
    "本周完成了 ResNet-50 迁移学习，准确率 87%"
)
# 输出：10 页 PPT 结构
```

---

### 4. 🔒 本地优先 + 局域网协作

**解决的问题：**
> 云端工具数据不可控，本地共享太麻烦。

**功能：**
- 数据完全本地存储
- 局域网 P2P 同步（Phase 3）
- 三级权限（公开/组内/私密）

**技术实现：**
- SQLite 本地数据库
- 加密存储（AES-256）
- 局域网发现（mDNS）

## 五、参赛材料

### 作品简介（150 字）

> 学术龙虾 v2 是一款科研知识管理工具，专为研究生、青年教师设计。核心功能包括：知识图谱（自动构建"问题 - 方法 - 结论"知识树）、智能推荐（实验 - 文献自动关联）、PPT 一键生成（聚合实验和论文）。所有数据本地存储，安全可控。目标是让碎片化的科研知识形成体系，让重复劳动自动化，让研究者专注于真正的创新。

### 核心价值

| 维度 | 说明 |
|------|------|
| 社会价值 | 促进科研公平，让资源有限的课题组也能获得智能支持 |
| 实际用处 | 知识图谱节省 50% 文献整理时间，智能推荐节省 30% 论文查找时间 |
| 创新性 | 知识图谱、实验 - 文献关联、PPT 一键生成均为独家功能 |
| 安全合规 | 数据完全本地，开源可审计，无隐私泄露风险 |

**演示（90 秒）：**
