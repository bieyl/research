# 学术龙虾 v2 - 科研知识大脑

## 项目概述

**学术龙虾 v2** 是一款面向科研工作者的 AI 研究助手，专为中关村北纬龙虾大赛・学术赛道设计。产品采用本地优先架构，保障科研数据安全，同时集成 arXiv 实时论文抓取，帮助研究者高效获取领域经典与前沿进展。

---

## 核心功能

### 1. 🌳 知识图谱

构建论文与实验的关联网络，让知识产生连接。

- **树状视图** - 清晰展示论文库与实验库
- **网络视图** - 可视化论文与实验之间的关联关系
- **智能关联** - 自动识别论文与实验的关键词、方法关联
- **去重逻辑** - 基于标题/目的的自动去重，避免重复录入

**技术实现：**
- 本地 SQLite 存储（可扩展 Redis 缓存）
- 基于关键词重叠、方法相似度、同领域识别的自动关联算法
- 支持树状/网络两种视图模式

---

### 2. 💡 智能推荐

**三轨制推荐策略** - 经典 + 前沿 + 实时

| 类型 | 标识 | 年份范围 | 说明 |
|------|------|---------|------|
| 📜 经典奠基 | 橙色 | 2012-2017 | 领域开山之作，必读基础 |
| 🆕 最新进展 | 绿色 | 2018-2023 | 前沿 SOTA，把握方向 |
| 🌐 arXiv 实时 | 粉色 | 实时抓取 | 最新预印本，抢占先机 |

**推荐模式：**
- **混合模式** - 本地演示 + arXiv 实时（默认）
- **仅本地** - 离线环境使用
- **仅实时** - 获取最新研究进展

**覆盖领域：**
- Transformer / 注意力机制
- CNN / 卷积神经网络 / 图像分类
- BERT / NLP / 语言模型
- 注意力机制（SENet、CBAM 等）
- ResNet / 残差网络

**经典论文代表：**
- Attention Is All You Need (2017) - Transformer 开山之作
- AlexNet (2012) - 深度学习革命开端
- ResNet (2016) - 152 层网络训练突破
- BERT (2019) - 11 项 NLP 任务 SOTA

**最新研究代表：**
- Swin Transformer (2021) - 视觉 SOTA
- MAE (2022) - 自监督学习新范式
- GPT-3 (2020) - 1750 亿参数语言模型
- LLaMA (2023) - 开源大模型代表

---

### 3. ⚡ 文献摘要

快速提取论文核心信息，5 秒读懂一篇论文。

**输出结构：**
- 【研究问题】- 论文解决的核心问题
- 【研究方法】- 采用的技术方法
- 【核心结论】- 主要研究发现
- 【创新点】- 与已有工作的区别
- 【关键词】- 3-5 个核心关键词

**支持场景：**
- 论文标题快速摘要（基于内置知识库）
- 论文内容粘贴摘要（基于 NLP 提取）

---

### 4. 📊 实验日志

自动生成结构化实验报告，让实验记录更规范。

**输出内容：**
- 【实验目的】- 实验要解决的问题
- 【实验条件】- 温度、压力、时间等参数
- 【实验结果】- 结构化表格呈现
- 【结果分析】- 趋势分析、异常值识别
- 【后续建议】- 优化方向与下一步实验建议

**示例：**
```
输入：温度 60/70/80 度，收率 45%/62%/38%

输出：
【实验目的】探究温度对反应收率的影响
【实验条件】温度梯度：60°C, 70°C, 80°C
【实验结果】
| 温度 (°C) | 收率 (%) |
|-----------|----------|
| 60        | 45       |
| 70        | 62       |
| 80        | 38       |
【结果分析】70°C 时收率最高 (62%)，温度过高可能导致副反应增加
【后续建议】建议在 65-75°C 范围内进一步细化温度梯度
```

---

### 5. 📑 PPT 大纲

一键生成学术汇报 PPT 框架，节省准备时间。

**生成结构：**
1. 研究背景与意义
2. 相关工作
3. 方法与技术路线
4. 实验设计与结果
5. 结论与展望
6. 参考文献

**支持场景：**
- 组会汇报
- 学术会议
- 论文答辩
- 项目评审

---

### 6. 📖 参考文献格式化

支持主流引用格式，一键生成规范参考文献。

**支持格式：**
- GB/T 7714（中国国家标准）
- APA（美国心理学会）
- MLA（现代语言协会）
- IEEE（电气电子工程师学会）
- BibTeX（LaTeX 标准）

**示例（GB/T 7714）：**
```
He K, Zhang X, Ren S, et al. Deep residual learning for image recognition[C]//Proceedings of the IEEE conference on computer vision and pattern recognition. 2016: 770-778.
```

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | HTML5 + CSS3 + JavaScript | 响应式设计，渐变配色，卡片式布局 |

| **数据存储** | SQLite | 本地存储，无需配置，开箱即用 |
| **外部 API** | arXiv API | 实时论文抓取，获取最新研究 |
| **NLP** | 自定义算法 | 关键词提取、相似度计算、摘要生成 |

### 核心模块

| 模块 | 文件 | 功能 |
|------|------|------|
| 知识图谱 | `knowledge_graph.py` | 论文/实验管理、关联识别、图谱可视化 |
| 智能推荐 | `smart_recommend.py` | 本地推荐、关键词匹配、相似度计算 |
| arXiv 抓取 | `arxiv_fetcher.py` | 实时论文搜索、经典/最新筛选 |
| 文献摘要 | `summarizer.py` | 结构化摘要生成 |
| 实验日志 | `lab_log.py` | 实验报告生成、数据分析 |
| PPT 大纲 | `ppt_outline.py` | 学术汇报框架生成 |
| 参考文献 | `reference_formatter.py` | 多格式引用生成 |
| Web 应用 | `web_app_v3.py` | 高端学术风格 UI |

## 使用场景

### 场景 1：文献调研

**问题：** 刚进入新领域，不知道从哪些论文开始读？

**解决方案：**
1. 输入领域关键词（如"Transformer 图像分类"）
2. 获取推荐论文（经典 + 前沿 + 实时）
3. 阅读经典论文建立知识框架
4. 追踪最新研究把握前沿方向

**效果：** 1 小时完成原本需要 1 周的文献调研

---

### 场景 2：实验记录

**问题：** 实验数据散乱，难以整理成规范报告？

**解决方案：**
1. 输入实验数据（如"温度 60/70/80 度，收率 45%/62%/38%"）
2. 自动生成结构化实验报告
3. 包含结果分析与后续建议

**效果：** 5 分钟完成原本需要 1 小时的实验记录

---

### 场景 3：组会汇报

**问题：** 明天组会，今晚才开始准备 PPT？

**解决方案：**
1. 输入研究主题
2. 一键生成 PPT 大纲
3. 填充具体内容

**效果：** 10 分钟完成 PPT 框架，专注内容填充

---

### 场景 4：论文写作

**问题：** 参考文献格式混乱，反复调整？

**解决方案：**
1. 输入论文信息
2. 选择目标格式（GB/T 7714、APA 等）
3. 一键生成规范引用

**效果：** 1 秒完成原本需要 5 分钟的格式调整

## 安装与部署

### 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/academic-lobster/v2.git
cd academic-lobster

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动服务
python src/web_app_v3.py

# 4. 访问界面
# 浏览器打开：http://localhost:5001
```

### 系统要求

| 要求 | 说明 |
