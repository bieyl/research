# 学术龙虾 v2 升级计划

## 目标
1. 实现实时论文抓取（arXiv API）
2. 前端界面高端学术风格优化

## 步骤

### Phase 1: 实时论文抓取模块
- [ ] 安装 arXiv API 客户端库
- [ ] 创建 `arxiv_fetcher.py` 模块
- [ ] 实现经典论文检索（高引用）
- [ ] 实现最新论文检索（按时间排序）
- [ ] 集成到推荐 API

### Phase 2: 前端界面优化
- [ ] 设计高端学术风格配色方案
- [ ] 优化卡片式布局
- [ ] 添加论文标签（经典/前沿/实时）
- [ ] 添加引用数显示
- [ ] 添加 arXiv 链接
- [ ] 优化响应式设计

### Phase 3: 测试与验证
- [ ] 测试 arXiv API 调用
- [ ] 验证论文数据准确性
- [ ] 浏览器 UI 测试
- [ ] 性能优化

## 时间估算
- Phase 1: 30 分钟
- Phase 2: 40 分钟
- Phase 3: 15 分钟
- 总计：~85 分钟

## 技术选型
- **arXiv API**: `arxiv` Python 库
- **前端**: CSS3 + 渐变 + 阴影 + 动画
- **配色**: 深蓝/紫色系（学术高端感）
