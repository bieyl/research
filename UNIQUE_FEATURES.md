# 🦞 学术龙虾核心差异化特色

## 一、市面上没有的功能（独家）

### 1. 🌳 科研知识图谱

**痛点：**
> 研究生读了 3 年论文，毕业时问他"这个领域有哪些研究方向？"，他说不清楚。因为知识是碎片化的，没有形成体系。

**现有工具：**
- Zotero/Mendeley：只是文献管理，无法形成知识网络
- ChatPDF：单次对话，无法积累
- 笔记软件（Notion/Obsidian）：需要手动建立关联

**学术龙虾方案：**
```
自动构建"问题 - 方法 - 结论"三层知识树：

研究问题：图像识别准确率提升
├── 方法分支 1：更深的网络
│   ├── ResNet (2016) → 残差连接 → 可训练 152 层
│   ├── DenseNet (2017) → 密集连接 → 参数效率提升
│   └── 你的实验 → 改进的残差块 → 准确率 87%→89%
├── 方法分支 2：注意力机制
│   ├── SENet (2018) → 通道注意力
│   └── CBAM (2018) → 空间 + 通道注意力
└── 方法分支 3：数据增强
    ├── Mixup (2017) → 图像混合
    └── CutMix (2019) → 图像裁剪混合
```

**技术实现：**
- 自动提取论文的"问题 - 方法 - 结论"三元组
- 基于关键词、方法相似度自动建立关联
- 可视化展示知识树和网络

---

### 2. 💡 实验 - 文献智能关联

**痛点：**
> 写论文时，忘记哪篇论文支持这个实验结果；做实验时，想不起哪篇论文提出了这个方法。

**现有工具：**
- 无。这是学术龙虾首创。

**学术龙虾方案：**
```
【录入实验】2026-03-12
实验：ResNet-50 改进，引入注意力机制，准确率 87%→89%

🤖 智能推荐相关论文：
1. ResNet 原始论文 (He et al., 2016)
   相关性：95 分 - 高度相关
   关联点：你的实验基于该论文的残差结构

2. Identity Mappings (He et al., 2016)
   相关性：88 分 - 中度相关
   关联点：该论文的"pre-activation"与你的改进相似

3. SENet (Hu et al., 2018)
   相关性：82 分 - 中度相关
   关联点：注意力机制的开创性工作

📝 引用建议：
建议在论文"方法"部分引用 [1] 说明基础架构，
在"讨论"部分引用 [2][3] 对比你的改进。
```

**技术实现：**
- 提取实验描述中的关键词和方法
- 与知识图谱中的论文进行相似度匹配
- 生成关联说明和引用建议

### 3. 📊 组会 PPT 一键生成

**痛点：**
> 每周组会要花 2-3 小时准备 PPT，但内容都是已有的实验记录和阅读的论文。

**现有工具：**
- Gamma/Tome：通用 PPT 生成，不懂科研场景
- 手动制作：耗时耗力

**学术龙虾方案：**
```
选择时间范围：2026-03-05 ~ 2026-03-12

自动生成 12 页 PPT：
P1 标题页 - 研究进展汇报 (2026.03.05-03.12)
P2 本周概览 - 3 个实验，5 篇论文，1 个突破
P3-5 实验进展 - 自动插入实验数据表格和趋势图
P6-8 文献阅读 - 自动插入论文摘要和知识图谱位置
P9 问题讨论 - 从实验记录的"问题"字段提取
P10-11 下周计划 - 基于未完成实验推荐
P12 致谢

导出格式：PPTX / PDF / Keynote
```

**技术实现：**
- 聚合指定时间范围内的实验记录和论文
- 自动生成数据图表（Matplotlib）
- 填充 PPT 模板（python-pptx）

### 4. 🔒 本地优先 + 局域网协作

**痛点：**
> 云端协作工具（Notion/飞书）数据不可控，本地共享又太麻烦。

**现有工具：**
- 云端：数据安全担忧
- 本地：无法协作

**学术龙虾方案：**
```
张师兄的电脑 (192.168.1.100)
├── 公开：课程笔记、教程 → 实验室所有人可见
├── 组内：实验记录、组会 PPT → 课题组成员可见
└── 私密：未发表数据、论文草稿 → 仅自己可见

你的电脑 (192.168.1.101)
├── 同步张师兄的"公开"和"组内"内容
├── 自己的数据三级分类
└── 离线可用，联网自动同步

数据始终存储在各白电脑，只同步变更元数据。
```

**技术实现：**
- 基于 P2P 的局域网发现
- 增量同步（只传变更）
- 三级权限控制

## 二、中文科研场景深度优化

### 1. GB/T 7714 参考文献格式

**痛点：** 中国高校毕业论文要求 GB/T 7714 格式，但现有工具（EndNote/Zotero）配置复杂。

**学术龙虾方案：**
```
输入：He K, Zhang X, Ren S, Sun J. Deep Residual Learning for Image Recognition. CVPR 2016.

一键输出：
【GB/T 7714】[1] HE K, ZHANG X, REN S, et al. Deep Residual Learning for Image Recognition[C]//CVPR, 2016.
【APA】He, K., Zhang, X., Ren, S., & Sun, J. (2016). Deep Residual Learning for Image Recognition. CVPR.
【IEEE】[1] K. He et al., "Deep Residual Learning for Image Recognition," CVPR, 2016.
```

---

### 2. 中文论文结构模板

**痛点：** 英文写作工具（Grammarly）不懂中文论文结构。

**学术龙虾方案：**
```
中文论文结构：
1. 摘要（300-500 字）
2. 引言（研究背景 + 问题定义 + 本文贡献）
3. 相关工作（按方法分类，而非按时间）
4. 方法（整体架构 + 关键模块）
5. 实验（数据集 + 对比方法 + 结果分析）
6. 结论（总结 + 局限 + 未来方向）

每部分提供写作要点和常用句式。
```

---

### 3. 中文学术用语润色

**痛点：** 直译英文导致中文论文读起来别扭。

**学术龙虾方案：**
```
原文：我们用了一个新的方法去做图像分类

润色：本文提出了一种新颖的图像分类方法

原文：这个结果比之前的方法都好

润色：实验结果表明，本方法在各项指标上均优于现有方法
```
