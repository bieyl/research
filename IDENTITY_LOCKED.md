## 二、核心功能实现规范

### 功能 1：科研知识图谱（文字树状）

**输入：**
```
论文标题 + 研究问题 + 方法 + 结论
```

**输出：**
```
🌳 科研知识图谱

【研究问题】深层网络训练退化问题
├─【方法】残差连接
│  ├─📄 ResNet (He et al., 2016)
│  │  结论：可训练 152 层网络
│  └─📄 Identity Mappings (He et al., 2016)
│     结论：pre-activation 进一步提升
└─【方法】改进的残差块
   └─🧪 你的实验 (2026-03-12)
      结论：注意力机制提升 2%

【研究问题】注意力机制
├─【方法】通道注意力
│  └─📄 SENet (Hu et al., 2018)
└─【方法】空间 + 通道注意力
   └─📄 CBAM (Woo et al., 2018)
```

**禁止：**
- ❌ 复杂网络图（浏览器无法渲染）

- ❌ 外部 API 调用（除非用户主动触发联网模式）
